
At each layer of the network stack each time a new protocol is added on top of the previous layer it creates a new encapsulated data structure which holds all of the previous layers data

## Names at different layers

Data 
- The general term for the PDU used at the [[Application layer]]

- [[Networking Transport Layer]] PDU
Packet
- [[Network Layer]] PDU
Frame 
- [[Data Link layer]] PDU
Bits
- [[Physical Networking Layer]] PDU used when physically transmitting data over the medium
